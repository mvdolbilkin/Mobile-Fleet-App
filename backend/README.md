# Модули Backend (internal/)

| Модуль | Описание |
|---|---|
| `auth` | Авторизация через Yandex Fleet API, сохранение WebView-сессий |
| `staff` | Управление водителями: список, профиль, статусы, массовые операции |
| `session` | Хранение сессий в Redis |
| `vehicles` | Транспортные средства |
| `expenses` | Расходы и транзакции |
| `fines` | Штрафы |
| `garage` | Гараж |
| `fleetmap` | Карта парка |
| `summary` | Сводная статистика |
| `competitions` | Соревнования |
| `goals` | Цели |
| `mailings` | Рассылки |
| `work_rules` | Условия работы |
| `menu` | Меню |
| `shared/middleware` | Auth middleware (проверка X-Park-ID) |
| `shared/proxy` | Прокси-утилиты к Yandex Fleet API |
